Hosting

Diese Seite wird über GitHub Pages gehostet (GitHub, Inc., 88 Colonial Center Parkway, Suite 100, Lakeland, FL 33803, USA). Beim Aufruf verarbeitet GitHub automatisch technische Server-Logs (u. a. IP-Adresse, Datum/Uhrzeit, aufgerufene Datei, Browser-Kennung). Details siehe die Datenschutzerklärung von GitHub.

Kartenkacheln (OpenStreetMap)

Die Kartendarstellung lädt Kartenkacheln direkt von OpenStreetMap-Servern nach – dabei wird deine IP-Adresse an OpenStreetMap übertragen. Das ist technisch notwendig, um die Karte darzustellen, und passiert bei jedem Seitenaufruf. Details siehe die Datenschutzerklärung der OpenStreetMap Foundation.

Ortssuche (Nominatim)

Die Funktion „Ort finden" schickt deine Sucheingabe an den Nominatim-Suchdienst von OpenStreetMap – dabei werden Sucheingabe und IP-Adresse an OpenStreetMap übertragen. Das passiert nur, wenn du die Suche aktiv nutzt.
